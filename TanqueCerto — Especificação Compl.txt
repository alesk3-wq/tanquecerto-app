# TanqueCerto — Especificação Completa do Sistema

## Visão Geral

O TanqueCerto é uma plataforma colaborativa para motoristas avaliarem postos de combustíveis com foco em:

* qualidade do combustível
* confiança do posto
* relatos de problemas mecânicos
* reputação da gasolina/etanol/diesel
* experiência de atendimento
* localização inteligente por GPS

A proposta principal é criar uma comunidade onde os próprios usuários ajudam outros motoristas a evitar combustíveis adulterados e encontrar postos confiáveis.

# Banco de Dados

## Tabela: users

```sql
CREATE TABLE users (
  id INT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  email VARCHAR(150) NOT NULL UNIQUE,
  password VARCHAR(255) NOT NULL,
  phone VARCHAR(20) NULL,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

---

## Tabela: stations

```sql
CREATE TABLE stations (
  id INT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(150) NOT NULL,
  brand VARCHAR(100),
  latitude DECIMAL(10,8) NOT NULL,
  longitude DECIMAL(11,8) NOT NULL,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

---

## Tabela: reports

```sql
CREATE TABLE reports (
  id INT AUTO_INCREMENT PRIMARY KEY,
  user_id INT NOT NULL,
  station_id INT NOT NULL,
  type ENUM('good', 'suspect', 'bad') NOT NULL,
  description TEXT,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

  FOREIGN KEY (user_id) REFERENCES users(id),
  FOREIGN KEY (station_id) REFERENCES stations(id)
);
```

# Sistema de Reputação

## Lógica Atual

### Relato positivo

```txt
+2 pontos
```

### Relato suspeito

```txt
-1 ponto
```

### Relato negativo

```txt
-2 pontos
```

---

## Resultado da reputação

### GOOD

Posto confiável.

### SUSPECT

Posto com relatos mistos.

### BAD

Posto com muitos relatos negativos.

### UNKNOWN

Sem relatos suficientes.
